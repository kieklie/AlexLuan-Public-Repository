# 栾宜伟 - 个人简历网站

一个基于纯前端技术构建的个人简历展示网站，部署在 GitHub Pages 上。

## 技术栈

- **HTML5** — 语义化页面结构
- **CSS3** — 渐变主题、动画效果、响应式布局
- **Vanilla JavaScript** — 粒子动画、滚动触发、3D 交互、证书轮播等

零框架依赖，无需构建工具。

## 页面板块

| 板块 | 说明 |
|------|------|
| Hero 首屏 | 动态渐变背景 + 粒子动画 + 打字机效果 |
| 关于我 | 个人信息卡片（毛玻璃风格） |
| 教育背景 | 时间线布局 |
| 工作经历 | 3D 倾斜卡片 + 彩虹流光边框 |
| 项目经历 | 网格瀑布流 + 翻转卡片 |
| 技能与证书 | 进度条动画 + 证书轮播/灯箱 |
| 页脚 | 联系方式 + 回到顶部 |

## 本地预览

直接在浏览器中打开 `index.html` 即可预览，无需启动任何服务。

## 部署

1. 创建 GitHub 仓库
2. 推送代码到 `main` 分支
3. 在仓库 **Settings → Pages** 中选择 `main` 分支
4. 访问 `https://<用户名>.github.io/<仓库名>/`

## 目录结构

```
Resume/
├── index.html          # 主页面（单页应用）
├── css/
│   └── style.css       # 样式表
├── js/
│   └── main.js         # 交互逻辑
├── assets/
│   └── images/         # 图片资源（头像、证书截图）
├── plan.md             # 建设方案
├── README.md           # 本文件
└── .gitignore
```
